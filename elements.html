VidesiVilla

This project is a comprehensive property management platform designed for seamless user experiences and efficient data handling. Key features include secure user authentication with JWT and Bcrypt, enabling sign-up and log-in capabilities. Users can create new listings with detailed information, manage options with easy selection and deselection features, and effortlessly handle photos with drag-and-drop, upload, and delete functions. Bookings are managed through an interactive calendar system, while property feeds are dynamically fetched by category. A robust search function allows keyword-based property searches, and users can manage favorites through an add/remove wishlist feature. Image storage is supported locally and integrated with MongoDB, utilizing Mongoose for schema management. The frontend is powered by Redux for state management, and styled with Sass (scss) and Material UI, ensuring both performance and aesthetic appeal.

TIC TAC TOE GAME

The Tic Tac Toe Game is a desktop-based application developed in Java, showcasing my expertise in GUI development and logical programming. A real-time clock at the bottom of the interface adds an engaging touch. This project highlights my skills in event handling, threading, and creating intuitive user experiences, reflecting my ability to design practical and enjoyable software solutions.
